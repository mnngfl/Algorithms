## 문제
#### 문제 설명
2016년 1월 1일은 금요일입니다. 2016년 a월 b일은 무슨 요일일까요? 두 수 a ,b를 입력받아 2016년 a월 b일이 무슨 요일인지 리턴하는 함수, solution을 완성하세요. 요일의 이름은 일요일부터 토요일까지 각각

`SUN,MON,TUE,WED,THU,FRI,SAT`

입니다. 예를 들어 a=5, b=24라면 5월 24일은 화요일이므로 문자열 TUE를 반환하세요.

#### 제한 조건
- 2016년은 윤년입니다.
- 2016년 a월 b일은 실제로 있는 날입니다. (13월 26일이나 2월 45일같은 날짜는 주어지지 않습니다)

#### 입출력 예
| a | b |	result |
|:-|:-|:-:|
| 5 |	24 |	"TUE" |

## 풀이
요일은 금요일부터 시작하므로 출력에 필요한 요일명 배열도 금요일부터 시작하도록 선언하였다.
또한 연도는 윤년인 2016년으로 고정되어 있으므로 해당 년도의 월별 마지막 날짜를 배열로 나타내 보면 다음과 같다.

``` js
var day_names = ['FRI', 'SAT', 'SUN', 'MON', 'TUE', 'WED', 'THU']; // 1월 1일(금요일) 부터의 요일명을 나타내는 배열
var days_in_month = [31, 29, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]; // 각 달의 마지막 날을 나타내는 배열
```
이후 입력으로 받은 `a월 b일`이 1월 1일부터 총 며칠이 흘렀는지를 계산하여 풀었다.

## 다른 풀이
자바스크립트의 `Date` 객체를 활용하면 각 달의 마지막 날짜를 몰라도 문제 풀이가 가능하다.

`Date.prototype.getDay()` 메서드는 주어진 날짜의 현지 시간 기준 요일을 반환한다. 이 때 반환 값은 0부터 6까지의 값이며 이 0은 일요일을 나타낸다.
주어진 날짜로 Date 객체를 생성하는 방법으로 문제를 풀어 보면 아래와 같이 작성할 수 있다.

``` js
function solution(a, b) {
    var day_names = ['SUN', 'MON', 'TUE', 'WED', 'THU', 'FRI', 'SAT'];
    var date = new Date(2016, a-1, b);
    return day_names[date.getDay()];
}
```

주의할 점은 Date 객체를 생성하는 과정에서 두 번째 인자로 전달된 `monthIndex`의 값이다. 이는 0이라는 값이 전달되면 1월을 뜻하고 11이 전달되면 12월을 뜻한다는 점에 유의하여야 겠다.
